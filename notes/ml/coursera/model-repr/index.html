    Model Function¶

    以上面的回归模型为例，为了方便理解，我们先将其近似地看作一元线性回归模型。

    基于先前在回归模型概述中的那幅图，我们不难得出一元线性回归模型的数学函数定义:

    $$ f_{w, b}(x^{(i)}) = wx^{(i)} + b $$

    显然，不同的 $w$ 和 $b$ 值会得到不同的直线:

    In [140]:
    Copied! 
    w = 0.6135
b = -1160

print(f"w = {w}")
print(f"b = {b}")
    w = 0.6135 b = -1160 print(f"w = {w}") print(f"b = {b}") 
    w = 0.6135
b = -1160

    通过 for 循环计算函数输出:

    • (ndarray (m,)) 表示一个含有 $m$ 个条目的 NumPy $n$ 维数组

    • (scalar) 表示一个标量

    • np.zero(n) 将返回一个包含 $n$ 条目的 NumPy 一维数组

    In [141]:
    Copied! 
    def compute_model_output(x, w, b) -> np.ndarray:
    """Compute the prediction of a linear model

    Args:
        x (ndarray (m,)): Data, m examples 
        w, b (scalar)    : model parameters  
    Returns
        f_wb (ndarray (m,)): model output
    """
    m = x.shape[0]
    f_wb = np.zeros(m)
    for i in range(m):
        f_wb[i] = w * x[i] + b
    return f_wb
    def compute_model_output(x, w, b) -> np.ndarray: """Compute the prediction of a linear model Args: x (ndarray (m,)): Data, m examples w, b (scalar) : model parameters Returns f_wb (ndarray (m,)): model output """ m = x.shape[0] f_wb = np.zeros(m) for i in range(m): f_wb[i] = w * x[i] + b return f_wb
