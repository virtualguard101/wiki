# 图的存储及基本操作

| | 邻接矩阵 | 邻接表 | 十字链表 | 邻接多重表 |
|:-:|:-:|:-:|:-:|:-:|
| 空间复杂度 | $O(\|V\|^2)$ | 无向图: $O(\|V\| + 2\|E\|)$ <br> 有向图: $O(\|V\| + \|E\|)$ | $O(\|V\| + \|E\|)$ | $O(\|V\| + \|E\|)$ |
| 找相邻边 | 遍历对应行或列的时间复杂度为 $O(\|V\|)$ | 找有向图的入度必须遍历整个邻接表 | 很方便 | 很方便 |

| 适用场景 | 稠密图 | 稀疏图和其他 | 只能存有向图 | 只能存无向图 |
| 表示方式唯一性 | 唯一 | 不唯一 | 不唯一 | 不唯一 |
