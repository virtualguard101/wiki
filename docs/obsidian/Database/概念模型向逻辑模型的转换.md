# 概念模型向逻辑模型的转换

完成概念模型向逻辑模型的转换，需要解决以下两个问题:

- 如何将实体型和实体间的联系转换为关系模式

- 如何确定这些关系模式的属性和码

E-R图转换为关系模型的实质就是将实体、实体的属性和实体之间的联系转换为关系模式。

## 实体的转换

- 原则

    - 一个实体对应一个关系模式

    - 实体的名称即是关系模式的名称

    - 实体的属性就是关系模式的属性

    - 实体的码就是关系模式的码

- 注意的问题

    - 属性域的问题: DBMS是否支持某些属性域

    - 非原子属性的问题
