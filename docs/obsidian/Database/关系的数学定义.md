# 关系的数学定义

## 基本概念

关系数据结构的形式化定义依赖于以下三个基本概念:

- **域**（*Domain*）

    - 域是一组具有相同数据类型的值的集合，记作 $D_i$。

    - 在关系模型中，使用域来表示实体属性的取值范围。

- [**笛卡儿积**（*Cartesian Product*）](../Math/DiscreteMath/集合%20&%20数学符号与逻辑命题.md#笛卡尔积)

    - 笛卡尔积可以表示为一个二维表，表中的每一个行对应一个元组，表中的每一个列对应一个域。

- [**关系**（*Relation*）](关系模型.md#关系)

    - 关系为**多个域的笛卡尔积**的**有限子集**，是一张二维表。

    - $D_1 \times D_2 \times \cdots \times D_n$ 的子集叫作在域 $D_1, D_2, \cdots, D_n$ 上的**关系**，记作 $R(D_1, D_2, \cdots, D_n)$。其中 $R$ 为关系名，$n$ 为关系的**目或度**（*Degree*）。

## 其他概念

- [**元组**（*Tuple*）](关系模型.md#元组)：笛卡尔积中的一个元素 $(d_1, d_2, \cdots, d_n)$，其中 $d_i \in D_i$，称为一个 $n$ 元组。

- **分量**（*Component*）：元组中的一个元素 $d_i$，称为一个分量。

- **基数**（*Cardinality*）：关系中元组的个数，称为关系的基数。

    - $D_1 \times D_2 \times \cdots \times D_n$ 的基数为 $|D_1| \times |D_2| \times \cdots \times |D_n| = \prod_{i=1}^n |D_i|$。
